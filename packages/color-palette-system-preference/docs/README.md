# Linking a color palette to the user's system preference

This example shows how to link each color palette to the user's system
preference for light and dark appearance.

Download example as archive:

[Download .zip][Download]{ .md-button .md-button--primary }

Unzip archive, change to the folder and enter:

``` bash
pip install -r requirements.txt
mkdocs serve
```
